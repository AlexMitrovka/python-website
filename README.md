# e-python-math: аналіз АКБ (Streamlit)

`app.py` — аналіз розряду з CSV тестера: метрики, оцінки 1–10, загальний Score за **оцінками** (не сирими значеннями), вердикт, C-rate, опційно напруга при 50% номіналу.

## Як запустити

1. Встановіть залежності:
   - `pip install -r requirements.txt`
2. Запустіть застосунок:
   - `streamlit run app.py`
3. В браузері відкрийте URL, який покаже `streamlit` (зазвичай `http://localhost:8501`).

## Що вміє

- Зчитує CSV, де є рядок заголовка типу `DATE,VOLTAGE(V),CURRENT(A),...`
- Автоматично знаходить рядок з `DATE` та ділить колонки по роздільнику `,` або `;`
- **Аналіз розряду** (налаштовується в боковій панелі):
  - старт розряду: перший рядок з `|I| ≥ I_threshold`
  - вікна **до/після старту** (за замовчуванням 2 с / 2 с)
  - відсічка: перша точка з `U ≤ V_cutoff` (за замовчуванням 3.2 V)
  - таблиця: стартова U, U відсічки, струм, mAh, Wh, середня U під навантаженням, просадка, внутрішній опір (оцінка), час тесту
  - **Номінал АКБ**: вручну (mAh) або **вибір моделі iPhone** (автопідстановка mAh); Wh_номінал = вручну або mAh×3.7/1000
  - **оцінка C-rate** за номіналом mAh
- Три графіки: **U vs E_CAPACITY** (інтервал тесту), **U vs час**, **I vs час** (з лініями старт/відсічка)
- **Оцінки 1–10** та **загальний бал (5/5 компонентів)**: `(mAh + Wh×1.5 + U×1.5 + ΔV×2 + R×2) / 8` після задання номіналу; у таблиці — формат на кшталт `2258 mAh (82%)`
- Додатково: довільні графіки (вибір X/Y)
